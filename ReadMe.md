# Wedding — Next.js 16

This project has been converted to **Next.js 16** (App Router, React 19).

## Quick start

1. **Copy static assets** (once):
   ```bash
   npm run copy-assets
   ```
2. **Install and run**:
   ```bash
   npm install
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` — Start development server (run `npm run copy-assets` first if `public/` is empty)
- `npm run build` — Copy assets and build for production
- `npm run start` — Start production server
- `npm run copy-assets` — Copy `css/`, `images/`, `fonts/` into `public/`

## Routes

- `/` — Home
- `/about` — Story
- `/services` — Services
- `/gallery` — Gallery
- `/contact` — Contact
